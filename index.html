Video Performance System

Viper is an interactive realtime image processing system developed using the Processing language and environment. It includes support for multiple channel image/video processing through OSC instructions to the Viper server.

Sample Instructions

To place an image:

{
    "method": "create",
    "posX": 35,
    "posY": 120,
    "image": "fish2.gif",
    "id": 1
}

This image file must be in the Viper projects data folder. Similarly, to place a video clip replace "image": "fish2.gif" with "video": "your-video-name.mp4". The ID field must be set by the client. Any further processing instruction on an already placed image or video will be referred by that media's ID. For example, to invert the image we just placed:

  {
    "id": 1,
    "method": "update",
    "invert": 1    
  }
